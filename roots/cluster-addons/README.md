# cluster-addons/

App-of-apps directory for the `cluster-addons-root`. Drop an `Application` or
`ApplicationSet` here to install an add-on (sealed-secrets, an ingress
controller, cert-manager, metrics, …) across your connected clusters; it runs in
the `cluster-addons` project.

This template ships **none** — add your own. (The course's Module 3 builds
sealed-secrets and ingress-nginx examples here.)
